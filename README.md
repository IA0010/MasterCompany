# MasterCompany

Sistema de gestión de empleados. API REST en .NET 10 + frontend en Angular.

## Tecnologías
- Backend: .NET 10, ASP.NET Core Web API
- Frontend: Angular 19
- Documentación: Swagger

## Arquitectura en capas
- MasterCompany.API - Controllers y configuración
- MasterCompany.Business - Lógica de negocio
- MasterCompany.Data - Acceso a datos

## Cómo ejecutar

### Backend
1. Abrir MasterCompany.sln en Visual Studio
2. Ejecutar con F5
3. Swagger en http://localhost:5115/swagger

### Frontend
1. Abrir la carpeta MasterCompany.Web
2. Ejecutar npm install y luego ng serve
3. Abrir http://localhost:4200
